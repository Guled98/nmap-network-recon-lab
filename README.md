# Nmap Network Recon Lab

This repository documents a basic network reconnaissance lab using Nmap for security analysis and penetration testing practice.

## Overview

The purpose of this lab is to demonstrate how Nmap can be used to gather information about a target system, including:

- Open ports
- Running services
- Service versions
- Operating system details

This type of reconnaissance is an important first step in both penetration testing and defensive security analysis.

## Lab Goals

- Understand basic Nmap scanning techniques
- Identify open ports on a target
- Detect services and versions
- Explore operating system detection
- Document findings clearly

## Lab Structure

- `nmap_basic_scan.md` – Basic host and port discovery
- `nmap_service_scan.md` – Service and version detection
- `nmap_os_detection.md` – Operating system detection
- `screenshots/` – Screenshots from the lab environment

## Tools Used

- Nmap
- Kali Linux
- Local lab / test environment

## Why this project

Network reconnaissance is a core skill in cybersecurity.  
This lab demonstrates practical use of Nmap as part of a structured learning process in penetration testing and network security.
